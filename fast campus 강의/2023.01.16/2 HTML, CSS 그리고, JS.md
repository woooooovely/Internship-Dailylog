# 2. HTML, CSS 그리고, JS

# 📌 HTML (기획자)

<aside>
💡 Hyper Text Markup Language.
페이지의 제목, 문단, 표, 이미지, 동영상 등 **웹의 구조를 담당**

</aside>

# 📌 CSS (디자이너)

<aside>
💡 Cascading Style Sheets.
실제 화면에 표시되는 방법(색상, 크기, 폰트, 레이아웃 등)을 지정해 **콘텐츠를 꾸며주는 시각적인 표현(정적)을 담당.**

</aside>

# 📌 JS (개발자)

<aside>
💡 Javascript.
콘텐츠를 바꾸고 **움직이는** 등 페이지를 동작시키는 **정적이 아닌 동적 처리를 담당**

</aside>
